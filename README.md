# 心の航行士 | Emotion Navigator

心の航行士（Emotion Navigator）是一個幫助使用者整理情緒、辨識內在狀態，並透過引導式流程逐步縮小情緒範圍的前端應用程式。  
本專案以 React + Vite + TypeScript 開發，重視沉浸感、清晰的互動流程，以及在桌面與手機裝置上的良好使用體驗。

---

### 專案簡介

這是一個以「情緒探索」為核心的互動式 Web App。  
使用者可以透過簡潔的介面與分階段引導，逐步辨識、縮小並理解自己當下的情緒狀態。

## 日本語紹介

### プロジェクト概要

**心の航行士（Emotion Navigator）** は、ユーザーが自分の感情を整理し、内面の状態をより明確に把握するためのフロントエンド Web アプリです。  
段階的なガイドを通して、曖昧な感情を少しずつ言語化し、より具体的に認識できるよう設計されています。

このプロジェクトの主な特徴：

- 感情を段階的に整理するガイド型フロー
- 中国語・日本語・英語の多言語対応
- デスクトップとモバイルの両方に対応したレスポンシブデザイン
- 静かで洗練されたブランド体験を意識した UI / UX
- React + Vite + TypeScript によるモダンなフロントエンド構成

### 技術スタック

- React
- Vite
- TypeScript
- CSS / フロントエンドスタイリング
- GitHub Pages（デプロイ）

## English Introduction

### Project Overview

**Emotion Navigator** is a guided front-end web application designed to help users clarify, narrow down, and better understand their emotional state.  
Through a step-by-step interactive flow, the app supports emotional reflection in a calm, structured, and accessible way.

Key highlights of this project:

- Guided emotional exploration flow
- Multilingual interface support (Chinese / Japanese / English)
- Responsive design for both desktop and mobile devices
- Clean and brand-oriented UI / UX
- Built with a modern front-end stack using React + Vite + TypeScript

### Tech Stack

- React
- Vite
- TypeScript
- CSS / Front-end styling
- GitHub Pages for deployment
